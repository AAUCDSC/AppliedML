Gradient Descent¶

In this section, you will fit the linear regression parameters to our dataset using gradient descent. The objective of linear regression is to minimize the cost function: $$J(\mathbf{\theta_0}) = \frac{1}{2m} \sum_{i=1}^{m}\left(h_\theta({x}^{(i)}) - y^{(i)}\right)^2$$

where the hypothesis $h_{\theta}(x))$ is given by: $$h_{\theta}(x) = \theta_0 + \theta_1x = \theta^{T}x$$

One way to do the parameters $\theta_j$ values is to use the batch gradient descent algorithm. In batch gradient descent, each iteration performs the update.

def gradientDescent(X, y, theta=[[0],[0]], alpha=0.01, num_iters=1500):
    m = y.size
    J_history = np.zeros(num_iters)
    for iter in np.arange(num_iters):
        # YOUR CODE HERE
        Y_pred = theta[1]*X[:,1] + theta[0]
        D_m = (1/m) * np.sum(np.multiply(X[:,1], (Y_pred-y[:,0])))
        D_c = (1/m) * np.sum(Y_pred-y[:,0])
        theta[1] = theta[1] - alpha * D_m
        theta[0] = theta[0] - alpha * D_c 
        J_history[iter] = computeCost(X,y[:,0], theta)
   #raise NotImplementedError()
    return(theta, J_history)
