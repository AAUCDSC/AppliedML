Gradient Descent¶

In this section, you will fit the linear regression parameters θ to our dataset using gradient descent. The objective of linear regression is to minimize the cost function: $$J(\mathbf{\theta_0}) = \frac{1}{2m} \sum_{i=1}^{m}\left(h_\theta({x}^{(i)}) - y^{(i)}\right)^2$$

where the hypothesis $h_{\theta}(x))$ is given by: $$h_{\theta}(x) = \theta_0 + \theta_1x = \theta^{T}x$$

One way to do the parameters $\theta_j$ values is to use the batch gradient descent algorithm. In batch gradient descent, each iteration performs the update.

def gradientDescent(X, y, theta=[[0],[0]], alpha=0.01, num_iters=1500):
    m = y.size
    J_history = np.zeros(num_iters)
    for iter in np.arange(num_iters):
        # YOUR CODE HERE
        # raise NotImplementedError()
        h = X.dot(theta)
        theta = theta - alpha * (1/m) * (X.T.dot(h-y))
        J_history[iter] = computeCost(X,y,theta)
    return(theta, J_history)
